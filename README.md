# 🎬 Netflix Clone

A front-end **Netflix Clone** project built using **HTML**, **CSS**, and **JavaScript**. This project replicates the look and feel of the Netflix landing page with modern UI elements and responsive design. Built for practice and educational purposes.

---

## 🌟 Features

- 📺 Stunning Netflix-like UI
- 📱 Fully Responsive Design
- 🖼️ Banner Section with Hero Image
- 🎞️ Movie Rows with Scrollable Thumbnails
- 🔥 Hover Effects and Transitions
- 🧪 Built for learning HTML, CSS, and JS basics

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3** (Flexbox, Grid, Media Queries)
- **JavaScript (ES6)**
